# Pollution Prediction and Health Advisory System

## Overview
This project predicts Air Quality Index (AQI) using Machine Learning techniques and visualizes pollution data using maps and graphs.

## Features
- AQI Prediction
- Pollution Analysis
- Interactive Pollution Map
- Health Advisory System
- Data Visualization

## Technologies Used
- Python
- Pandas
- Scikit-learn
- XGBoost
- Streamlit
- Folium

## Files
- app.py → Streamlit application
- Pollution_model.ipynb → ML model training
- map_visualization.ipynb → Map visualization
- delhi_ncr_aqi_dataset.csv → Dataset

## Run Project
```bash
streamlit run app.py
